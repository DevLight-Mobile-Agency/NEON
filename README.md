NEON
====

The sample open-source player which include music visualizations and equalizer.

The application contains following visualizations:

 - [**SlideAlbumCover**](#cover).
 - [**BarGraph**](#bar).
 - [**GravityDefied**](#gd).
 - [**SeaBreath**](#sea).
 - [**Pulse**](#pulse).
